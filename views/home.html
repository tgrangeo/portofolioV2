Your Gateway to My Projects

This site began as a personal portfolio and has grown into a dynamic GitHub profile viewer. By using the GitHub API, it automatically displays the README files of my projects, giving a clear view of the work I’ve done and the skills I bring to the table.

It also lets you explore other GitHub profiles—just type a username into the search bar to see their repositories and project READMEs. When profiles are well-documented, the experience is even better, making it easy to dive into their work.

This project reflects my commitment to improving as a developer and my interest in showcasing the work of others in the open-source community. My aim is to provide a useful tool while also demonstrating my technical skills to senior developers and recruiters.
